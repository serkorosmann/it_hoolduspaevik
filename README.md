# IT-hoolduspäevik

See programm on loodud IT-hooldustööde ja paranduste dokumenteerimiseks. Rakendus võimaldab kasutajal kirjeid lisada, otsida, filtreerida ja salvestada andmeid JSON-vormingus.

## Funktsionaalsus
* Kirjete lisamine (pealkiri min 4 tähemärki, kirjeldus min 10).
* Kirjete kuvamine ja kustutamine.
* Otsing märksõna järgi.
* Filtreerimine staatuse (OPEN/DONE) järgi.
* Andmete automaatne salvestamine `logbook.json` faili.

## Kasutamine
Programmi käivitamiseks käivita fail `main.py`:
```bash
python main.py
